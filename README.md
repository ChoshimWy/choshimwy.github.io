# Choshim Wei · Indie Apps

Multi-app personal developer site. Each app has its own product page and privacy policy under `apps/`.

## Structure

- `index.html`: Homepage
- `app-ads.txt`: App-ads declaration
- `apps/noctibeat/`: NoctiBeat website and privacy policy
- `apps/formulavis/`: FormulaVis website, privacy policy, and formula library JSON

## Apps

- **NoctiBeat**
  - Website: `/apps/noctibeat/index.html`
  - Privacy: `/apps/noctibeat/privacy.html`
  - App Store: https://apps.apple.com/cn/app/%E5%A4%9C%E5%BE%8B/id6758320663
- **FormulaVis**
  - Website: `/apps/formulavis/index.html`
  - Privacy: `/apps/formulavis/privacy.html`
  - Formula library: `/apps/formulavis/data/formula_library.json`
